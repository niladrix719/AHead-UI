
# Ahead UI

A UI website of Ahead dribbble APP.


## Run Locally

Clone the project

```bash
  git clone https://github.com/niladrix719/AHead-UI.git
```

Go to the project directory

```bash
  cd AHead-UI
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Tech Stack

Next.js, Typescript, TailwindCSS, GSAP
